Graphic Design · Typography · 2024

Dialogue:
Basel

Type Coursework / Study Abroad
Location Basel, Switzerland
Tools Illustrator · InDesign · p5.js

Overview

Basel has a particular claim on the history of graphic design — the Swiss Style, the International Typographic Style, and the work of designers like Armin Hofmann and Emil Ruder all trace back to the city's design schools. This project was developed during a course there, working within and against that tradition.

The work explores typography as structure: how letterforms can be treated as compositional objects, how grids can be followed and broken deliberately, and how visual tension is created through proximity, weight, and white space.

The goal wasn't to replicate Swiss Style — it was to understand it well enough to make meaningful departures from it.

Approach

The project unfolded across several studies. The first focused on pure typographic composition — setting text at different scales, weights, and orientations to understand how type creates movement across a page without any imagery. The second introduced a system: a consistent visual language that could be applied across multiple formats and still read as coherent.

A third component brought the work into motion and code — translating the static compositions into generative p5.js sketches that used the same typographic logic but introduced time as a design variable.

Reflection

What this project made clear is that constraint and creativity aren't opposites. Working within a tight typographic grid — and understanding exactly why each rule exists — makes breaking it more intentional and more effective. That principle transfers directly to product design: knowing manufacturing constraints deeply is what makes it possible to design around them cleverly.
